# WSN Guardian

White-label security response platform for South African security companies.

## Overview

WSN Guardian is a modern emergency response platform that allows security companies to offer their clients a mobile app for instant emergency assistance. The platform includes:

- **Client Mobile App (PWA)** - Emergency panic buttons with GPS tracking
- **Control Room Dashboard** - Real-time incident management (coming soon)
- **White-Label Ready** - Customizable per security company
- **Firebase Backend** - Scalable, real-time database
